# State Street SST

Editor support for [State Street](https://joshabracks.github.io/State-Street/) template files —
`.sst.ts` and `.sst.js`.

## Features

- **Embedded HTML highlighting.** The HTML inside your State Street template literals is highlighted
  as HTML (Markdown too), instead of one flat string.
- **Dedicated languages** for `.sst.ts` and `.sst.js`, each with its own **file icon** (blue for
  TypeScript, yellow for JavaScript).

## About State Street

State Street is a lightweight, zero-dependency reactive single-page-app framework: components are
plain functions that return HTML strings, and a single reactive state object drives dependency-gated
re-renders. Scaffold a project with `npm create @state-street`.

- Docs: https://joshabracks.github.io/State-Street/
- GitHub: https://github.com/Joshabracks/State-Street
